Build your own neuronal network, and train it with your data!

This website is primarily about data.
I have tried to map the typical process a Data Scientist goes through on this website.
Therefore there are three features:
The first feature is data sourcing, where you can generate any Twitter data and stock data 60 days back, that is then stored in your database.
The second feature is data analysis, where you can see correlations between your data with the help of an LSTM model, in short a small AI.
The third feature is data visualization, where you can visualize your data any way you want.
I hope that the application can help you with your datasience project.
